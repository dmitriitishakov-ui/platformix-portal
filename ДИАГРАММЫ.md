# Диаграммы проекта

## Архитектура

Как устроено приложение: браузер делает запросы к Flask, тот проверяет JWT-токен
для защищённых операций и работает с базой SQLite. Swagger документирует API.

```mermaid
flowchart TD
    A[Браузер<br/>HTML / CSS / JS Fetch] -->|HTTP-запросы JSON| B[Flask-сервер app.py<br/>Маршруты REST API]
    B -->|документация| S[Swagger /apidocs]
    B -->|проверка токена| C[JWT-авторизация<br/>защита POST /documents]
    C -->|SQL-запросы| D[(SQLite portal.db<br/>documents, users)]
```

## Схема базы данных

Две таблицы: документы и пользователи (для авторизации).

```mermaid
erDiagram
    documents {
        int id PK
        string title
        string category
        string author
        string content
        string created_at
    }
    users {
        int id PK
        string username
        string password_hash
    }
```
